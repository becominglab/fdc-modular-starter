# Workshop カリキュラム設計

**目標**: founders-direct-modular をステップバイステップで構築しながら、モダンWeb開発のベストプラクティスを習得する

---

## 全体構成

| モジュール | 内容 | 所要時間 | 習得スキル |
|-----------|------|---------|-----------|
| 0 | 環境構築 | 30分 | Terminal, Node.js, Git |
| 1 | プロジェクト初期化 | 45分 | Next.js, TypeScript |
| 2 | 認証フロー | 60分 | 状態管理, ルーティング |
| 3 | ダッシュボード | 45分 | コンポーネント設計, Context |
| 4 | タスク管理 | 60分 | CRUD, useReducer |
| 5 | 設定ページ | 45分 | フォーム, バリデーション |
| 6 | リード管理 | 90分 | モーダル, 検索・フィルタ |
| 7 | デプロイ | 30分 | Vercel, 環境変数 |

**合計**: 約7時間（分割受講可）

---

## UI/UX ベストプラクティス

### 画面遷移パターン

- ページ遷移: メインナビゲーション (Link, router.push)
- モーダル: 作成/編集フォーム、確認ダイアログ
- タブ: 同一コンテキスト内の切り替え

### レイアウト構成

- Header（固定）+ Navigation Tabs + Main Content Area
- 業界標準のSaaSダッシュボードパターン

---

## 各モジュールの学習目標

### Module 0-1: 環境構築 & プロジェクト初期化
- ターミナル操作、Node.js、Git
- Next.js App Router、TypeScript strict mode

### Module 2-3: 認証 & ダッシュボード
- React Context で状態管理
- コンポーネント分割の考え方

### Module 4-5: タスク管理 & 設定
- useReducer でデータ管理
- フォームバリデーション

### Module 6-7: リード管理 & デプロイ
- モーダル、検索・フィルタ
- Vercel デプロイ
