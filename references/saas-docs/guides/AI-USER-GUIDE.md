## 1. AI機能の概要

### 1.1 AI機能でできること

FDC（Founders Direct Cockpit）のAI機能は、営業活動を支援する以下の機能を提供します：

| 機能 | 説明 |
|------|------|
| **営業アドバイス** | 見込み客の状況に応じた営業戦略のアドバイス |
| **メッセージ生成** | 顧客への連絡文・メールの下書き生成 |
| **提案書支援** | 提案内容の整理・構成サポート |
| **タスク優先度提案** | 効果的なタスクの優先順位づけ |
| **データ分析** | 営業データの傾向分析・インサイト |

### 1.2 利用できるモデル

- **GPT-4o-mini**（推奨）: 高速・低コスト、日常的な質問に最適
- **GPT-4o**: より高度な分析・長文生成に最適

### 1.3 利用制限

| 項目 | 制限 |
|------|------|
| リクエスト数 | 月間1,000回（デフォルト） |
| トークン数 | 月間500,000トークン |
| コスト上限 | 月間$10.00 |

※ 管理者により調整可能です。

## 2. 利用開始方法

### 2.1 AI機能の有効化

AI機能を利用するには、管理者による設定が必要です。

1. **管理者に連絡**: ワークスペースの管理者にAI機能の有効化を依頼
2. **利用規約の確認**: [AI利用規約](/ai-terms) を確認
3. **設定完了**: 管理者がAPIキーを設定後、利用可能に

### 2.2 AI機能へのアクセス

AI機能が有効な場合、以下の場所からアクセスできます：

- **ダッシュボード**: 右下のAIアシスタントボタン
- **見込み客詳細**: 「AIに相談」ボタン
- **タスク一覧**: 優先度提案機能

## 3. AIアシスタントの使い方

### 3.1 基本的な質問の仕方

AIアシスタントには自然な日本語で質問できます。

**効果的な質問例:**

```
✅ 良い例:
- 「株式会社ABCへの初回コンタクトメールを作成してください」
- 「今月の商談で重視すべきポイントは何ですか？」
- 「製造業向けのアプローチ方法を教えてください」

❌ 改善が必要な例:
- 「メールを書いて」（対象が不明確）
- 「売上を上げたい」（質問が抽象的すぎる）
```

### 3.2 コンテキストの活用

AIは以下の情報を参照して回答を生成します：

- **ビジネス情報**: MVV、LeanCanvas、OKR
- **顧客情報**: 見込み客・既存客のデータ
- **タスク情報**: 進行中のタスク・目標

コンテキストレベルを調整することで、参照する情報量を制御できます。

### 3.3 会話の継続

AIアシスタントは会話の文脈を理解します。

```
ユーザー: 「株式会社ABCへのメールを作成して」
AI: [メール案を生成]
ユーザー: 「もう少しカジュアルな表現にして」
AI: [修正版を生成]
ユーザー: 「冒頭に季節の挨拶を追加」
AI: [さらに修正版を生成]
```

## 4. コンテキストレベル

### 4.1 コンテキストレベルとは

AIが参照する情報の範囲を設定できます。レベルが高いほど詳細な回答が可能ですが、トークン使用量が増加します。

| レベル | 参照情報 | 推奨用途 |
|--------|---------|---------|
| **minimal** | 基本情報のみ | 一般的な質問 |
| **standard** | ビジネス情報 + 顧客概要 | 日常的な営業相談 |
| **detailed** | 詳細な顧客データ含む | 個別案件の相談 |
| **full** | 全情報（タスク・履歴含む） | 総合的な分析 |

### 4.2 レベルの選び方

```
📋 一般的な営業ノウハウの質問
   → minimal または standard

📊 特定の顧客への提案相談
   → detailed

📈 営業戦略の総合的なアドバイス
   → full
```

### 4.3 設定方法

1. **設定** > **セキュリティとプライバシー** > **AI設定** にアクセス
2. デフォルトコンテキストレベルを選択
3. 保存

## 5. テンプレート機能

### 5.1 テンプレートとは

よく使うメッセージの雛形を保存・再利用できる機能です。

### 5.2 変数プレースホルダー

テンプレートには変数を埋め込めます。実行時に自動で置換されます。

| 変数 | 説明 | 例 |
|------|------|-----|
| `{{顧客名}}` | 顧客の担当者名 | 山田様 |
| `{{会社名}}` | 会社名 | 株式会社ABC |
| `{{担当者名}}` | 自分の名前 | 鈴木 |
| `{{今日}}` | 今日の日付 | 2025年12月2日（火） |
| `{{来週}}` | 来週の日付 | 12月9日（火） |

### 5.3 テンプレートの作成

1. **設定** > **テンプレート管理** にアクセス
2. 「新規作成」をクリック
3. テンプレート名、カテゴリ、内容を入力
4. 変数パレットから変数を挿入
5. 保存

### 5.4 テンプレートの使用

1. AIアシスタントでテンプレートを選択
2. 変数が自動で置換される
3. 必要に応じて編集
4. 送信またはコピー

## 6. ベストプラクティス

### 6.1 効果的なAI活用のコツ

1. **具体的に質問する**
   - 対象顧客、目的、制約条件を明確に

2. **段階的に詳細化する**
   - 大まかな方向性 → 詳細な内容 の順に

3. **コンテキストレベルを適切に設定**
   - 必要以上に高くしない（コスト・速度に影響）

4. **出力は必ず確認・編集**
   - AIの出力はあくまで下書き
   - 事実確認・表現の調整を行う

### 6.2 やってはいけないこと

- **機密情報の直接入力**: 個人情報や機密情報は直接入力しない
- **AI出力の無確認使用**: 必ず内容を確認してから使用
- **過度な依存**: 最終判断は必ず人間が行う

## 7. よくある質問（FAQ）

### Q1. AI機能が使えません

**A:** 以下を確認してください：
1. AI機能が有効化されているか（管理者に確認）
2. 月間使用量の上限に達していないか
3. ブラウザのJavaScriptが有効か

### Q2. 回答が遅い

**A:** 以下をお試しください：
1. コンテキストレベルを下げる
2. 質問を短くする
3. ネットワーク接続を確認

### Q3. 回答の精度が低い

**A:** 以下をお試しください：
1. 質問をより具体的に
2. コンテキストレベルを上げる
3. 前提条件を明示的に伝える

### Q4. 使用量を確認したい

**A:** **管理** > **AI使用状況** で確認できます。

## 8. トラブルシューティング

### 8.1 エラーメッセージと対処法

| エラー | 原因 | 対処法 |
|--------|------|--------|
| 「AI機能はオフです」 | テナントでAI無効 | 管理者に有効化を依頼 |
| 「使用上限に達しました」 | 月間クォータ超過 | 翌月まで待つか、管理者に上限引き上げを依頼 |
| 「リクエストが多すぎます」 | レート制限 | 少し待ってから再試行 |
| 「回答を生成できませんでした」 | API エラー | 時間をおいて再試行 |

### 8.2 データが反映されない

1. ブラウザをリロード
2. ログアウト → 再ログイン
3. キャッシュをクリア

### 8.3 サポートへの連絡

解決しない場合は、管理者または サポート（support@foundersdirect.jp）にお問い合わせください。

## 9. 利用規約・プライバシー

### 9.1 AI利用規約

AI機能の利用に際しては、[AI利用規約](/ai-terms) に同意いただく必要があります。

主なポイント：
- AI生成コンテンツは参考情報であり、最終判断はユーザーが行う
- 機密情報・個人情報の取り扱いに注意
- 違法・不正な目的での利用は禁止

### 9.2 データの取り扱い

- AIへの入力データはOpenAI APIに送信されます
- 個人情報（メール、電話番号、氏名）は自動的にマスクされます
- AIとの会話履歴はワークスペースデータとして保存されます
